# Engineering Admission Predictor

A Laravel-based web application that helps students predict engineering college admissions using official Maharashtra CET Cell cutoff data for:

- First Year Engineering (FE)
- Direct Second Year Engineering (DSE)

Features:
- College Prediction
- Branch Filtering
- FE & DSE Support
- College Cutoff Search
- 2024 & 2025 Cutoff Analysis

Tech Stack:
- Laravel
- PHP
- MySQL
- Bootstrap 5
- Select2
